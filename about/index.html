Projects

A basic Compiler for simplified C language
Zhuolin Yang, 2017

Margatroid compiler is a basic compiler for M* language, mainly based on Java. Graph Coloring for Register Allocation and some other optimizers has been added. It contains 12000+ code and finished in a short time.

A Simple 5-stages pipelined MIPS microprocessor
Zhuolin Yang, 2017

This is a basic 5-stages pipelined MIPS microprocessor, using Harvard Structure. It's implemented by Verilog HDL, and finished in a week.

Implementation of Advanced Data Structures
Zhuolin Yang, Zhenjia Xu, Fangao Zeng, 2016

A Coursework of Data Structure 2016. I implemented three challenge Data structures: Strict Fibonacci Heap, AAA Tree, PQ Tree with my group consist of three of my classmates.
